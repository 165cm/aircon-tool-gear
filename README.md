# エアコン工具ギア

エアコン工具ギアは、エアコン修理・取付に必要な工具を、初心者にも分かりやすく、中級者以上にも納得感があるように整理した日本向けの工具比較サイトです。

公開サイト: <https://165cm.github.io/aircon-tool-gear/>

開発・運用の詳細は [Developer.md](./Developer.md) を参照してください。

## サイトの目的

エアコン職人を目指す人、副業で取付作業を始めたい人、DIYで最低限の工具を揃えたい人に向けて、道具選びの判断基準をまとめています。

価格の安さだけではなく、精度、耐久性、安全性、現場での使いやすさ、将来の買い替えまで含めて比較できる構成です。

## 運営方針

当サイトはアフィリエイト広告を利用しています。価格、在庫、仕様は変動するため、購入前にAmazonおよびメーカー公式情報を必ず確認してください。

本サイトの情報は、工具選びの比較・検討を補助するためのものです。施工、修理、冷媒回収などの作業は、関連法令、資格、メーカー指定手順、安全基準を確認したうえで行ってください。

## SEOと更新計画

GitHub Pagesで公開し、静的HTML、構造化データ、サイトマップ、RSSを生成しています。

繁忙期に向けて、予約記事をもとに週3回程度の更新ができる構成にしています。ロングテールキーワードとして、型番比較、工具の使い方、施工トラブル、季節需要に関する記事を追加していく方針です。
